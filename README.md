# sparfit
Abnehmen. Sparen. Einfach essen.  SparFit ist eine Web-App für Menschen, die gesund abnehmen und gleichzeitig ihr Lebensmittelbudget im Blick behalten möchten.  Die App kombiniert Essensplanung, Einkaufsliste, Kalorienkontrolle, Gewichtsverlauf und Budgetverwaltung in einer einfachen und übersichtlichen Lösung.  Funktionen  ✅
